## About
A todo app for creating and organizing tasks. Based on a JIRA board style layout. 

Uses Next.js, Typescript and Tailwind for app structure and styling.

React-Query is used for simplifying API calls and managing query data / state.

dnd-kit is used for drag and drop features.

## Getting Started

1. Clone the repository to your machine

2. Run `npm install` to install the dependencies

3. Rename `.env.sample` to `.env`

4. Ensure that the [backend](https://github.com/JayaDeHart/todo-backend/) is running on `localhost:8000`

5. Run `npm run dev` to start the project
